Multi-dimensional Information Security Ontology (MDISOnt) is a comprehensive information security ontology that helps security specialists and decision-makers remove ambiguous understanding of the information security domain, decompose security into several perspectives using dimensional views and modules, and efficiently manage information security within an organization.
